جمل التحكم الشرطي في بايثون

مقدمة

Conditional control statements are used to make decisions in programs based on certain conditions. In short, they are a way to direct the flow of a program depending on whether certain conditions are true or false.

جمل التحكم الشرطي تُستخدم لاتخاذ قرارات في البرامج بناءً على شروط معينة. باختصار، هي طريقة لتوجيه سير البرنامج اعتمادًا على ما إذا كانت شروط معينة صحيحة أو خاطئة.

تركيب جملة if الأساسية

مثال بسيط:

if x > 5:
    print("x greater than 5")

التفسير: إذا كان المتغير x أكبر من 5، سيتم طباعة "x greater than 5".

أمثلة

استخدام else مع if

if age < 18:
    print("You are a minor.")
else:
    print("You are an adult.")

التفسير: إذا كان العمر أقل من 18، سيتم طباعة "You are a minor"، وإلا سيتم طباعة "You are an adult".

استخدام elif

if mark >= 90:
    print("Excellent")
elif mark >= 75:
    print("Very Good")
elif mark >= 60:
    print("Good")
else:
    print("Fail")

التفسير: يتم اختبار الشروط بالترتيب، إذا كانت الدرجة بين 75 و89، سيتم طباعة "Very Good".

استخدام and و or

if age > 18 and age < 65:
    print("You are an adult but not a senior.")

Shorthand if-else

result = "Pass" if score >= 60 else "Fail"

Nested if

if age >= 18:
    if has_license:
        print("You can drive.")
    else:
        print("You need a license.")
else:
    print("You are too young.")

استخدام input

age = int(input("Enter your age: "))
if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

Password verification

password = input("Enter password: ")
if password == "secret123":
    print("Access granted!")
else:
    print("Access denied!")

تمارين

تمرين 1

اكتب برنامج يطلب من المستخدم عمره ويطبع "You are a minor" إذا كان أقل من 18، و"You are an adult" إذا كان 18 أو أكبر.

تمرين 2

اكتب برنامج يطلب من المستخدم درجة امتحان (من 0 إلى 100) ويطبع النتيجة كالتالي:

  • Excellent إذا كانت الدرجة 90 أو أعلى.
  • Very Good إذا كانت الدرجة بين 75 و89.
  • Good إذا كانت الدرجة بين 60 و74.
  • Fail إذا كانت الدرجة أقل من 60.

تمرين 3

اكتب برنامج يقارن بين رقمين يدخلهما المستخدم ويطبع أيهما أكبر أو إذا كانا متساويين.
